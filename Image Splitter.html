    ⌨️ Keyboard shortcuts & help

    • Ctrl+V — paste image from clipboard
    • Enter — split image (when controls focused)
    • D — download ZIP of all pieces
    • R — reset / load a new image
    • T — cycle theme (Shift+T = previous)
    • S — open settings
    • P — toggle results side panel
    • ? — show this help
    • Esc — close dialogs / lightbox / panel
    • ← → — navigate pieces in lightbox or tabs
    • Ctrl+A — select all pieces (when grid focused)

    📝 Naming pattern tokens

    • {prefix} — your file name prefix
    • {row} / {col} — zero-padded row / column number
    • {n} — sequential piece number
    • {rows} / {cols} — total rows / columns
    • {ext} — file extension

    💡 Tips

    • Auto-split is on by default — pieces are generated the moment you load an image and update live as you change settings.
    • Use the quick presets (2×2, 3×3, …) for one-click splitting.
    • Drop multiple images to batch process them sequentially.
    • Use Advanced options for rotation, flip, gap, and background color.
    • The grid overlay updates live as you change splitting options.
    • Click any piece to view it large; navigate with arrow keys.
    • Switch to compact view in the pieces section for many small thumbs.
