Authorized Security Testing Suite

A collection of non-destructive, offline security tools for lab environments, QA checks, and defensive validation. Everything runs locally in your browser, stores data locally, and requires no API key.

ETHICAL USE ONLY Use these tools only on systems you own or are explicitly authorized to test. All tools run locally in your browser — nothing is transmitted. Outputs are intentionally non-destructive and omit stealth, persistence, evasion, exploitation chains, and weaponization guidance.

    XSS Workbench
    
                        
    Build safe, marker-based reflection probes (with an inert render preview), audit a snippet for unsafe output handling, and grade a Content-Security-Policy — all locally. Nothing is executed or transmitted, and the payloads are benign (they only reveal where input is reflected).
    

                        
    
                            
                            
                            
                        
    

                        
                        
    
                            
                            

                            
                            

                            
                            

                            
                            

                            
    
                                Options
                                
                                
                                
                            
    

                            
    
                                
                                
                            
    

                            
    🔒 Probes are designed to be reflected verbatim so you can see the exact escaping applied. The benign style uses inert markers (no script execution); the detection style adds context-breakout characters (< > " ') to spot missing output encoding. Choose All contexts for a full matrix.
    
                        
    

                        
                        
    
                            
                            
                            
    
                                
                                
                                
                            
    
                            
    Heuristic static review: it flags dangerous DOM sinks (innerHTML, document.write, eval, setAttribute('href', …)), risky framework patterns (dangerouslySetInnerHTML, v-html, bypassSecurityTrust*, {{{…}}}), and javascript: URLs, then suggests safer patterns. It does not execute anything.
    
                        
    

                        
                        
    
                            
                            
                            
    
                                
                                
                                
                            
    
                            
    Parses a Content-Security-Policy value entirely offline and grades its resistance to XSS: it checks script-src for 'unsafe-inline'/'unsafe-eval', wildcards, missing object-src/base-uri, absent nonces/hashes, and more. Nothing is fetched or sent.
